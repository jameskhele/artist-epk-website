# CLAUDE.md – Shevyn Roberts EPK Website Build

## 🎯 Objective
Build a **premium, booking-ready artist EPK website** for Shevyn Roberts.

The final product must be:
- Clean, minimal, and highly professional
- Suitable for **booking agents, managers, labels, press, and award submissions**
- Comparable to **Bandzoogle-style EPKs**
- Completed within a **24-hour build scope**

---

## 🎨 Design Direction

### Style
- Clean, modern, elegant
- NOT flashy, cluttered, or experimental
- Premium, trustworthy, industry-standard

### Branding
- Primary Colors:
  - Teal
  - Hot Pink
  - Neon Purple
  - Black
  - White
- Use dark theme base with neon accents
- Strong typography hierarchy
- Large, high-quality imagery

### UX Principles
- Fast and frictionless navigation
- Clear calls-to-action (Booking, Listen)
- Mobile-first responsiveness
- Simple scroll-based layout
- No unnecessary animations

---

## 🏗️ Tech Requirements

- Framework: **Next.js (preferred) or React**
- Styling: Tailwind CSS or clean modular CSS
- Deployment: Vercel or Netlify
- Performance optimized
- SEO-ready metadata
- Accessible (contrast, buttons, readability)
